üå≥ IOTA Smart Contracts Initiative

The IOTA Smart Contracts Initiative is a collaborative effort to help people in the IOTA Community by introducing consistent workflows around IOTA ISCP with content, and open source contributions.

IOTA is an open-source distributed ledger technology that allows connected devices to transfer data and IOTA tokens among each other for zero fees.
